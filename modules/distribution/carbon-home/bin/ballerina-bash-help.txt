
Usage:
      ballerina.sh ballerina-file [args...]

        Executes the Ballerina main function in given Ballerina file
        (Eg: main.bal).

        Use args to defines input arguments for Ballerina main function.
        Arguments are separated by spaces. If argument contains a space, define
        the argument within double quote. (Eg: "Hello World")

   or ballerina.sh command

        where a command can be one of the following:
            version     Print Ballerina version.
            help        Print this help message.
